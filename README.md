## Features

- Lots of Foo.

## Building

Build the app via:

```console
$ npm run all
```

## Launching the app

Start an interpreter via e.g.:

```console
$ npm start lisp
$ npm start scheme
$ npm start prolog
```

## Exiting the app

Exit an interpreter by typing 'exit'.
